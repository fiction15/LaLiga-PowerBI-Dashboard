
# ⚽ La Liga Performance Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard built to analyze La Liga football performance across multiple seasons.

The dashboard combines traditional football statistics with advanced analytics to provide insights into team performance, scoring efficiency, possession impact, tactical formations, and match outcomes.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization
- Data Cleaning & Transformation

---

## Dashboard Features

### Executive Dashboard

- Total Matches KPI
- Total Goals KPI
- Total Wins KPI
- Total Points KPI
- Average Possession KPI
- Top Goal Scoring Teams
- Top Winning Teams
- Goals Trend by Season

### Advanced Analytics

- Expected Goals (xG) vs Actual Goals
- Possession vs League Points Analysis
- Most Effective Formations
- Home vs Away Performance
- Shot Accuracy KPI

---

## Dataset

- 4,700+ La Liga match records
- Team statistics
- Goals scored
- Possession %
- Expected Goals (xG)
- Tactical formations
- Match outcomes

---

## Key Insights

- Home teams generally perform better than away teams.
- Possession alone does not guarantee more points.
- Certain formations consistently produce better results.
- Teams convert approximately one-third of their shots into shots on target.
- Expected Goals (xG) helps evaluate finishing efficiency.
